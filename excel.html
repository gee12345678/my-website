This Excel dashboard project analyzes data from a survey on student social media usage. The purpose was to explore patterns of screen time, app preferences, and potential signs of digital addiction among students.

  • Created a dynamic dashboard using Pivot Tables, Slicers, and Pivot Charts

Tools & Features Used:

What I Learned:

  • Designing dashboards with a clean layout and storytelling approach
  • Using Excel to analyze survey data and draw actionable insights
  • Building interactive reports for non-technical stakeholders

Project Takeaway: This project taught me how to create user-friendly dashboards and communicate data insights using only Excel, which is a highly valuable tool in many business environments.
