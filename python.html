🏠 House Price Prediction (California Housing Dataset)

Objective: Predict housing prices using various features from the California housing dataset for a fictional real estate company, UrbanNest Realty.

🔍 What I Did:

  • Loaded and explored the California housing dataset from sklearn.
  • Visualized distributions, correlations, and relationships between features using scatter plots and heatmaps.
  • Preprocessed data, normalized features, and built two models: Linear Regression and Decision Tree Regressor.
  • Evaluated models using R² Score and Mean Absolute Error (MAE).

📈 Key Features:

  • Median Income: Strongly correlated with house value.
  • Location (Latitude, Longitude): Helped visualize price trends across California.
  • Rooms and Population per Household: Helped uncover local density and living space availability.

📊 Results:

  • Linear Regression: MAE ≈ $50,000, R² ≈ 0.61
  • Decision Tree: MAE ≈ $35,000, R² ≈ 0.75

💡 Key Insights:

  • Decision Trees perform better with non-linear data than simple linear regression.
  • Median income is a reliable indicator of house prices in California.
  • Location plays a crucial role in housing value and should not be ignored.

🛍️ Customer Segmentation (Mall Customers Dataset)

Objective: Segment customers into groups based on their spending score and income to help ShopEase create targeted marketing strategies.

🔍 What I Did:

  • Loaded and explored the Mall Customers dataset including Age, Gender, Annual Income, and Spending Score.
  • Cleaned the data (no missing values), visualized patterns using Seaborn and Matplotlib.
  • Applied k-Means clustering using Annual Income and Spending Score.
  • Determined the optimal number of clusters (k=5) using the Elbow Method.

📊 Clusters and Strategy:

  • Cluster 1: High income, high spenders → Premium promotion
  • Cluster 2: Low income, low spenders → Value offers
  • Cluster 3: High income, low spenders → Incentives or loyalty
  • Cluster 4: Low income, high spenders → Cautious targeting
  • Cluster 5: Average income and spenders → General campaigns

📈 Visuals:

  • Scatter plot of clusters with centroids.
  • Histogram distributions by gender and spending score.
  • Elbow plot to choose optimal k value.

💡 Key Insights:

  • Spending habits aren't directly proportional to income — segmentation reveals hidden groups.
  • Cluster analysis helps in personalizing marketing for different consumer types.
  • Gender does not significantly impact spending score distribution in this dataset.
