This was my first Tableau dashboard project, designed to explore global population trends across countries and continents. The main goal was to build a clean, interactive dashboard that presents basic demographic insights in a visually engaging and accessible format.

The dashboard allows users to view population totals by continent, compare country populations, and analyze how the population is distributed globally. It includes visual components such as bar charts, highlight tables, and maps for intuitive comparisons.
