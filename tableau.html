📊 My First Dashboard

🔍 Project Overview:
This dashboard provides a global perspective on key demographic indicators such as fertility rates, life expectancy, gender equality, and reproductive health. It enables users to explore population trends and disparities across different countries.

🎯 Project Goal:

  • To create an easy-to-use visual tool for exploring population statistics by region or country.
  • To help users understand the social, health, and economic factors influencing population changes.

📊 Tools & Techniques Used:

  • Tableau Desktop for building the dashboard
  • Filters and drop-down menus for interactivity
  • Line graphs, bar charts, and maps to compare population data
  • Color coding for demographic clarity (e.g., high vs. low fertility rates)

📂 Data Sources & Preparation:

  • Used global datasets (likely sourced from UN, World Bank, or public repositories)
  • Cleaned data to ensure accuracy, removed missing or null values
  • Standardized country names for consistency in mapping visuals

⚙️ Skills Demonstrated:

  • Data visualization and storytelling
  • Tableau dashboard design principles
  • Basic statistical interpretation
  • Filtering and parameter creation

🚧 Challenges Faced:

  • Aligning different datasets (e.g., merging population and fertility data)
  • Designing a user-friendly layout that works on various screen sizes

💡 What I Learned:

  • The importance of clean and consistent data
  • How to make data tell a story through charts
  • How demographic data can inform policy decisions

🌱 Future Improvements:

  • Add year-over-year comparison sliders
  • Include forecasts or projections using trend lines
  • Add tooltips that explain each metric for beginners

🎬 Netflix Dashboard

🔍 Project Overview:
This interactive Tableau dashboard analyzes Netflix's global content library by various dimensions such as genre, release year, ratings, and country of origin. It provides insights into what kinds of content are dominant and how Netflix's catalog has evolved.

🎯 Project Goal:

  • To explore patterns in Netflix content using visual data analysis.
  • To offer interactive features for filtering and analyzing user-preferred content.

📊 Tools & Techniques Used:

  • Tableau Desktop
  • Pie charts and bar graphs for categorical data
  • Heat maps and timeline visualizations
  • Interactive filters (genre, year, rating, country)
  • Use of calculated fields for data transformation (e.g., extracting year from release date)

📂 Data Sources & Preparation:

  • Sourced from Kaggle (Netflix Movies and TV Shows dataset)
  • Cleaned columns with inconsistent values (e.g., multiple genres)
  • Extracted relevant features: type (Movie/TV Show), country, duration, genre, release year

⚙️ Skills Demonstrated:

  • Categorical data grouping and filtering
  • Calculated fields and advanced filter controls
  • Dashboard layout and color usage for better UX

🚧 Challenges Faced:

  • Dealing with multi-valued genre fields
  • Visualizing complex categorical combinations without cluttering the dashboard

💡 What I Learned:

  • How to use filters effectively to allow user-driven exploration
  • The power of storytelling through visuals — for example, showing how certain genres became more popular over time
  • Gained confidence in using Tableau’s full dashboard capabilities

🌱 Future Improvements:

  • Add sentiment analysis of show reviews if data is available
  • Show trending genres over time using a line graph
  • Embed a search feature for specific titles
