📊 Job Salaries by Industry Dashboard

🔍 Project Overview:
This dashboard presents a comprehensive analysis of job salaries across various industries. It enables users to explore salary trends based on experience level, job title, location, and work type (remote/on-site), offering insight into compensation patterns within the job market.

🎯 Project Goal:

  • To build an interactive dashboard that highlights salary differences across industries and job roles.
  • To support job seekers, analysts, and HR teams in understanding how salaries vary with experience, region, and job type.

📊 Tools & Techniques Used:

  • Tableau Desktop for creating interactive visualizations
  • Filters and parameters for dynamic data exploration (e.g. by experience level, location, remote work)
  • Bar charts, box plots, and KPI summary tiles for quick comparison
  • Color-coded visuals to highlight top-paying industries

📂 Data Sources & Preparation:

  • Dataset: Data Science & Tech Job Salaries from Kaggle
  • Cleaned missing/invalid values, standardized job titles and industry names
  • Created calculated fields (e.g. average salary per industry)

⚙️ Skills Demonstrated:

  • Dashboard design and user-centered layout
  • Data cleaning and transformation
  • Using filters, parameters, and interactive charts in Tableau
  • Creating calculated fields and combining multiple insights into one view

🚧 Challenges Faced:

  • Normalizing job titles and locations across the dataset
  • Designing a layout that’s clean but still contains many insights

💡 What I Learned:

  • The power of data visualization in simplifying salary analysis
  • How to design with recruiters, HR teams, and job seekers in mind
  • The importance of choosing the right chart type to make comparisons clear

🌱 Future Improvements:

  • Include trend analysis over time if longitudinal data becomes available
  • Drill-down dashboards by job family or company size
  • Integrate external job market data to enrich analysis

🎬 Netflix Dashboard

🔍 Project Overview:
This interactive Tableau dashboard analyzes Netflix's global content library by various dimensions such as genre, release year, ratings, and country of origin. It provides insights into what kinds of content are dominant and how Netflix's catalog has evolved.

🎯 Project Goal:

  • To explore patterns in Netflix content using visual data analysis.
  • To offer interactive features for filtering and analyzing user-preferred content.

📊 Tools & Techniques Used:

  • Tableau Desktop
  • Pie charts and bar graphs for categorical data
  • Heat maps and timeline visualizations
  • Interactive filters (genre, year, rating, country)
  • Use of calculated fields for data transformation (e.g., extracting year from release date)

📂 Data Sources & Preparation:

  • Sourced from Kaggle (Netflix Movies and TV Shows dataset)
  • Cleaned columns with inconsistent values (e.g., multiple genres)
  • Extracted relevant features: type (Movie/TV Show), country, duration, genre, release year

⚙️ Skills Demonstrated:

  • Categorical data grouping and filtering
  • Calculated fields and advanced filter controls
  • Dashboard layout and color usage for better UX

🚧 Challenges Faced:

  • Dealing with multi-valued genre fields
  • Visualizing complex categorical combinations without cluttering the dashboard

💡 What I Learned:

  • How to use filters effectively to allow user-driven exploration
  • The power of storytelling through visuals — for example, showing how certain genres became more popular over time
  • Gained confidence in using Tableau’s full dashboard capabilities

🌱 Future Improvements:

  • Add sentiment analysis of show reviews if data is available
  • Show trending genres over time using a line graph
  • Embed a search feature for specific titles

📊 My First Dashboard

🔍 Project Overview:
This dashboard provides an analytical view of crime trends across various regions. It allows users to explore crime rates by type, region, and year, offering insight into safety and law enforcement issues in different areas.

🎯 Project Goal:

  • To create a user-friendly dashboard for visualizing crime statistics by category and location.
  • To help users, policymakers, and researchers identify crime hotspots and trends over time.

📊 Tools & Techniques Used:

  • Tableau Desktop for interactive dashboard creation
  • Filters and drop-down selectors for dynamic exploration
  • Bar charts, line graphs, and heat maps for visual comparisons
  • Color coding to highlight severity and frequency of crimes

📂 Data Sources & Preparation:

  • Crime dataset (from police reports, government sources, or open crime databases)
  • Cleaned and formatted to remove duplicates and inconsistencies
  • Standardized crime categories and location names

⚙️ Skills Demonstrated:

  • Data storytelling and visualization with Tableau
  • Designing interactive dashboards
  • Data cleaning and transformation
  • Creating calculated fields and filters

🚧 Challenges Faced:

  • Handling inconsistent reporting formats from different regions
  • Visualizing large datasets without cluttering the dashboard

💡 What I Learned:

  • The importance of data standardization in public datasets
  • How visuals can make crime data more accessible and actionable
  • The role of data visualization in supporting public safety initiatives

🌱 Future Improvements:

  • Add time sliders for year-over-year comparisons
  • Include predictive trends using statistical models
  • Add map tooltips with quick summaries for each region
